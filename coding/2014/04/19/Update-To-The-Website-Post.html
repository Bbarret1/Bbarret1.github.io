Update To The Website

Blog Home

19 Apr 2014

Hello Internet,

So I am updating my website because I have learned a lot since I began this self-taught coding experience. As I learn I want to incorporate everything that I like into this website, my personal website. I will be blogging about my coding experiences, along with some of my past experiences and probably explaining where I am going with it so that if anyone wanted to learn they could follow in my steps. I will also be blogging about what I am getting into, what I find interesting, and possibly other thoughts I have along the way.

I am now using Jekyll to create this website. It is a layout platform that allows you to make layouts of the website, and then if you want to create a new website, or edit something across all of the web pages you can do less editing and have your results faster. By this I mean the webpage you are reading this on is drawing from an html file in a layouts folder such that the footer and header are identical across every webpage. If I were to change that html file it would change every webpage across the entire website. Pretty neat huh? While it was a major pain to set up I really think Jekyll is a worthwhile investment (only time, it doesn't cost anything) if you have the time to learn it. This will make editing a website, or blogging much easier to do because you can focus on getting the change made rather than making sure every i is dotted or t is crossed, even though when coding there are not dots or crosses.

If you are reading this stick around for more thoughts a long the way or let me know if you have any thoughts yourself!
